Pertemuan 3: Struktur Percabangan

1. Konsep Percabangan

Percabangan memungkinkan program untuk mengambil keputusan berdasarkan kondisi tertentu. Jika kondisi terpenuhi (TRUE), program menjalankan blok kode A. Jika tidak (FALSE), program menjalankan blok kode B.

2. Struktur IF - ELSE

Struktur paling dasar dalam pemrograman.

Studi Kasus:
Menentukan kelulusan siswa. Jika nilai > 75 maka Lulus, jika tidak maka Remedial. 

int nilai = 80;

if (nilai > 75) {
    cout << "Selamat, Anda Lulus!";
} else {
    cout << "Maaf, Anda harus Remedial.";
}

3. Struktur Switch Case

Digunakan jika terdapat banyak pilihan kondisi yang nilainya tetap (bukan rentang).


int nomorHari = 1;

switch(nomorHari) {
  case 1:
    cout << "Senin";
    break;
  case 2:
    cout << "Selasa";
    break;
  default:
    cout << "Hari tidak valid";
}

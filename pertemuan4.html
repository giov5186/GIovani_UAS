Pertemuan 4: Struktur Perulangan (Looping)

1. Apa itu Looping?

Looping digunakan untuk menjalankan baris kode yang sama berulang kali selama kondisi tertentu terpenuhi. Ini menghemat waktu penulisan kode.

2. Struktur FOR Loop

Digunakan ketika kita sudah mengetahui berapa kali perulangan akan dilakukan.


// Mencetak angka 1 sampai 5
for (int i = 1; i <= 5; i++) {
    cout << "Angka ke-" << i << endl;
}

3. Struktur WHILE Loop

Digunakan ketika jumlah perulangan belum diketahui pasti, dan bergantung pada kondisi (True/False).


int i = 1;
while (i <= 5) {
    cout << "Perulangan While ke-" << i << endl;
    i++; // Increment agar tidak terjadi infinite loop
}

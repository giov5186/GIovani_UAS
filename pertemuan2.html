Pertemuan 2: Variabel, Tipe Data & Operator

1. Apa itu Variabel?

Bayangkan variabel sebagai sebuah wadah atau kotak untuk menyimpan data sementara di memori komputer. Setiap wadah harus diberi nama agar mudah dipanggil.

Rumus deklarasi: tipe_data nama_variabel = nilai;

2. Jenis Tipe Data Dasar

Tipe Data Fungsi Contoh
Integer (int) Angka bulat 10, -5, 100
Float / Double Angka desimal 3.14, 2.5
String Teks / Kalimat "Halo Dunia"
Boolean Kondisi benar/salah True, False

3. Operator Aritmatika

Digunakan untuk operasi matematika sederhana:

  • + : Penjumlahan
  • - : Pengurangan
  • * : Perkalian
  • / : Pembagian
  • % : Modulus (Sisa Bagi)

Contoh Kode (C++)


#include <iostream>
using namespace std;

int main() {
    int umur = 20;            // Tipe data Integer
    string nama = "Budi";     // Tipe data String
    
    cout << "Nama: " << nama << endl;
    cout << "Umur: " << umur << endl;
    
    return 0;
}

Pertemuan 5: Array (Larik)

1. Pengertian Array

Array adalah variabel spesial yang dapat menyimpan banyak nilai dalam satu nama variabel yang sama. Setiap nilai memiliki index (dimulai dari 0).

Analogi:
Jika Variabel adalah sebuah kotak sepatu, maka Array adalah sebuah lemari loker yang memiliki banyak pintu bernomor.

2. Deklarasi & Akses Array

Contoh membuat array berisi 4 nama mobil:


#include <iostream>
#include <string>
using namespace std;

int main() {
    // Membuat Array
    string cars[4] = {"Volvo", "BMW", "Ford", "Mazda"};
    
    // Mengakses data (Index mulai dari 0)
    cout << cars[0]; // Output: Volvo
    cout << cars[1]; // Output: BMW
    
    return 0;
}

3. Mengubah Isi Array


cars[0] = "Toyota";
cout << cars[0]; // Sekarang Outputnya: Toyota
